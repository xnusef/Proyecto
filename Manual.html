Manual


INICIO


Registro
Debes rellenar todos los campos con informacion valida para poder registrarte en la base de datos.
Login
Una vez registrado el usuario puedes iniciar sesion para poder acceder a las funciones del programa.


PROGRAMA


Funciones para oficinistas y gerentes


Reserva
En esta pestaña es donde se realizan las reservas hacia el cliente, una vez se completan todos los campos correspondientes, estos se envian a la base de datos en espera de ser pagados.

Modificar reserva
En caso de haber puesto un dato mal en la reserva, desde esta pestaña pueden modificarlos para luego reenviarlos.

Consultas
En esta pestaña puedes ver los datos de cada consulta y en caso de que esta sea erronea darle de baja.

Cobro
Aqui simplemente se cobran las reservas y se aprueban para ser enviadas definitivamente a la base de datos.

Recepcion y Retiro
Cuando la mascota llegue al Hotel se tiene que marcar desde esta pestaña para asi dar por terminado el proceso de inscripcion de esta. Tambien se usa para cuando se vaya a retirar a la mascota.

Funciones únicamente para gerentes


Modificar datos del hotel
Aqui se pueden dar de baja a un oficinista seleccionado y cambiar los precios del hotel.
